# Term-1-Uni-project

A quick, terminal based, text adventure game about escaping a prison/dungeon

### features:
- 2 different endings
- ASCII graphics for relevent sections
- item and inventory system
- visable text based map



### Starting the game: 
 - ensure both myGame.exe and text.txt are in the same directory 
 - open myGame.exe and follow the instructions as follows 
 
    -- wait until all text has finished displaying before pressing enter
    -- the terminal should automatically resize itself to the correct size
       so don't shrink the terminal size any more
